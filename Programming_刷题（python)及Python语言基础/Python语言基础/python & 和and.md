## python & 和and

&是逻辑与运算，当与的二者某位都是真(也就是1)的时候才能给该位赋1，其余情况是假（也就是0）。

and是逻辑判断，返回的是真（非0值）或者假（值0），在算式a and b中，只要a、b中有一个为0就返回0（假）值。否则就返回b的值。
