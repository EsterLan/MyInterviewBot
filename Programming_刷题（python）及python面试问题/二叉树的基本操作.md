## 二叉树的基本操作

### 二叉树的遍历

```python
class TreeNode:
    def __init(self, x):
        self.val = x
        self.left = None
        self.right = None
```



#### 先序遍历

T/S: O(n)/O(h) h为二叉树高度

```python
@param root TreeNode
@return int整型一维数组
def preoderTraversal(root):
    if not root:
        return []
    stack = [root]
    res = []
    while stack:
        node = stack.pop(-1)
        res.append(node.val)
        if node.right: # 右子节点先入栈后出栈
            stack.append(node.right)
        if node.left:
            stack.append(node.left)
    return res
```

#### 中序遍历

  1. 使用一个栈保存结点（列表实现）；
  2. 如果结点存在，入栈，然后将当前指针指向左子树，直到为空；
  3. 当前结点不存在，则出栈栈顶元素，并把当前指针指向栈顶元素的右子树；
  4. 栈不为空，则循环步骤2、3。

```python
def inOrder(root):
    if not root:
        return []
    stack = []
    p = root
    while p or stack:
        while p: # 所有左节点入栈
            stack.append(p)
            p = p.left
        if stack:
            p = stack.pop(-1)
            res.append(p.val)
            p = p.right
```



#### 后序遍历

```python
def PostOrder(root):
    stack = [root]
    stack2 = []
    result = []
    while len(stack) > 0:
        root = stack.pop(-1)
        stack2.append(root) # 根-右-左进栈
        if root.left:
            stack.append(root.left)
        if root.right:
            stack.append(root.right)
    while len(stack2) > 0:
        node = stack2.pop(-1)
        result.append(node.val)
    return result
```

def traversal(root):

### 层次遍历BFS

```python
def BFS(root):
    if not root:
        return []
    queue = [root]
    res = []
    while queue:
        node = queue.pop()
        res.append(node.val)
        if node.left:
            queue.append(node.left)
        if node.right:
            queue.append(node.right)
    return res
            
```

@return [list[int]]
def BFS_ceng(root):
    if not root:
        return []
    queue = [root]
    res = []
    while queue:
        next_level = []
        val = []
        while queue: # 取出当前层元素，并将下一层存入next_level
            node = queue.pop()
            val.append(node.val)
            if node.left:
                next_level.append(node.left)
            if node.right:
                next_level.append(node.right)
        queue = next_level
        res.append(val)
    return res
